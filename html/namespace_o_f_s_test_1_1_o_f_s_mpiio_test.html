def OFSTest.OFSMpiioTest.romio_testsuite (   testing_node,
  output = [] 
)

This is the romio testsuite that comes with mpich and can also be used with OpenMPI.

This does not use pbs/torque, but uses the MPI process managers directly.

Parameters
testing_nodeOFSTestNode on which tests are run.
outputArray that holds output from commands. Passed by reference.
Returns
0 Test ran successfully
Not 0 Test failed
